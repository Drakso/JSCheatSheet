Conditionals

A simple if conditional in javascript. It checks if the condition in the () are true and if so, it executes the code block inside the {}

A simple if/else conditional in javascript. If the condition in the if statement is not true than the block of code in else is executed.

If conditional can execute the code block if there is operation in the () and its true. Everything that is not true such as undefined or null executes the else code block.

if/else if statemets are used if there are more conditions dependent on eachother. When the first if conditional is not true it goes to the next and so on until one is true or it hits the else. Note that if two if conditions are true, only the first one will execute its codeblock. The others after that are skipped.

There can be if statements inside of other if statements.

A function can also be set as a condition for an if statement. The function return value will be used as the condition.

DOM Manipulation

Selectors are a very important part in javascript. They connect our javascript code with the HTML elements. The most commonly used selectors are by id, class and tag name. The selector by id returns an element while the selectors by class and tag name return collection of elements so you have to treat it like an array and use loops to find a certain element or change them all.

Query selectors are introduced later than the id, class and tag name selectors. They can select any element just like the css selectors. . is for classes, # is for id and you type the name of the html tag if you want to select a tag name. JQuery selectors work in a similar way.

An event listener waits for the user to interact in some way with an element or object and executes some code as a result of that. In the example, by clicking the button an event is trigerd on the click and an alert is shown on the screen.

JQuery event listeners do the same function as the pure javascript one. There are two types of jquery event listeners. The difference between the two is only in the conventions.

With the selectors, we can read value from elements such as the input element and use that value in our code. We use the element object property .value.

An important way of manipulation the DOM is by changing the contents of elements. This is done with the element object property .innerHTML. To clear an element just set the .innerHTML to an empty string "".

Css style can also be changed through javascript. All you have to do is select an element and access that element object property .style like in the example.

Adding or removing css classes from elements is also possible. The element object property .classList with the properties .add and .remove can add or remove element css classes. This is usefull for hiding elements at will, like the example shows.

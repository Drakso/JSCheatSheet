Functions

Function declaration is the most common way to create a function. Declared functions can be declared on their own but they must have a name

Functuon expressions are used when function is needed on the spot. They can't be created on their own and don't need to be named. A workaround for creating a function expression on its own is to wrap it with ()

Arrow functions are replacement for regular expressions in ES6. They are simpler and easier to use.

Functions can be passed as function arguments. The function passed as argument function is executed first.

Default parameters of a function can be set. Whenever an argument is not given for a parameter of a function, it will take the default value for that parameter if it is set. If default value is not set, the value of a parameter with no argument is set to undefined.

Recursion is when a function calls it self. Something to keep in mind for these functions is that they can easily get into an infinite loop, forever calling eachother which can result in the browser crashing or the page not loading.

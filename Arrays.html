Arrays

An array is a collection of items. You can put variables, numbers, strings, objects and even functions. Arrays are declared with the [ ] brackets. Arrays have properties that help us manipulate with them. Properties are accessed by adding a . next to the name of the array. The most common and useful property is the lengt property.

Elements are called by adding the index of the element in a [ ] brackets. Element indexes are starting from 0 so the first index will always be index 0. Elements can be rewritten at any time with new data. If we skip some indexes while declaring a new element, the skipped indexes will have a value of undefined .

Adding and removing items at the end of an array is done with the array properties push and pop. Simply type push and the item you want to add at the end of the array in ( ) brackets or pop and empty ( ) brackets to remove the last item from an array.

Adding and removing items at the start of an array is done with the array properties unshift and shift. Simply type unshift and the item you want to add at the beginning of the array in ( ) brackets or shift and empty ( ) brackets to remove the first item from an array.

Filtering an array is done with the array properties filter and find. Both of them check the items in the array by a criteria added in a callback function in ( ) brackets. Find returns all the items that passed the criteria in a new array, while find only returns the first item that meets the criteria and does not check the rest.

Running a particular function for every element is also possible with the forEach and map method. forEach runs a function using every element and does not return a new array as a result. map on the other hand runs a function on every element and returns a new array with the altered elements by the function.

Finding indexes of certain items can be achieved with the array properties findIndex and indexOf. findIndex needs a callback function to test the items in the array. As soon as it finds an item that passes the criteria it stops and returns its index. indexOf needs an item so it can match it with the elements in the array. When it finds a match it stops and returns the index. A second argument can be passed, a number, which represents the starting point of the search for a match.

An array property called join is used if there is a need for an array to be converted in to a string of its elements. If an argument is passed to the join property, that character or string will be the separator between the elements in the string. Default is ,. Reversing an array is also possible with the array property reverse, which reverses the elements in the array (Reverses the array, does not create a new array with reverse items).

Selecting a range of items from an array is done with the array properties slice and splice. slice requires the start and finish index, so that it can return a new array with the items between those indexes (it includes the starting index but not the finishing). splice also needs a start and finish index, but it removes the items from the existing array between those indexes (also includes the starting index but not the finishing) and it returns a new array with the removed items. You can also add new items after the finishing index. The new items are added from the starting index entered, forward, pushing all the other items to the right.

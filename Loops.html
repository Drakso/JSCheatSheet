Loops

Loops are used to automate repetitive tasks. The simplest loop is the while loop. For the while loop, a counter variable needs to be declared outside of the while block so it can remember the iterations. A condition in the ( ) is checked every iteration and if it is true, the loop continues to execute the code inside the block over and over until the condition is false

A common way of using loops is for traversing trough collections such as arrays. Inside of a loop we can put any code that we need. An if conditional, object, function and even another loop.

The for loop is the most widely used loop because it is shorter than the while. It is shorter because the declaration of the counter, the condition and the incrementing of the counter is all written as arguments in ( ) brackets.

Except the declaration of the counter and the incrementing, everything else is pretty much the same with the for loop as it is with the while. For loops are more widely used because they eliminate the chance of forgetting to increment the counter and creating an infinite loop that can possibly slow down or even crash your browser.

Loops usually go through all the code after every cycle and stop when the condition is not met any more. Break allows us to stop the execution at a certain point and to stop the loop from continuing with its iterations even if the condition in the loop is met. So when we put break, all the code below it, will not execute and the loop will stop iterating.

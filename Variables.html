Variables

These are the normal types of variable declaration. In javascript you can declare with var and let. In EcmaScript6 let was introduced which lives in the block scope vs the var that lives in the function scope.

Variables can be declared but not initialized. In that case the variable holds an undefined value and type. Variables can also be initialized with operations or other variables.

Declaration in multiple rows is done with ` ` and it is very usefull for writing HTML code in JavaScript

Multiple variables can be initialized just by separating them with a ,

If we redeclare a variable with the keyword var, it will not lose its value. For let that is not the case. Also in javascript, initialization goes from left to right( meaning operation on the left are executed first and on the right last )

Variables in javascript can handle all types even if they are combined. It handles that by turning everything into a string.

Javascript variables are dynamic, meaning that if you declare a number to a variable, making that variable with type number, you can reinitialize it with a string with no problems.

In javascript you can keep function in a variable. Then you can call the variable like its a function, and it will execute the function in it and return the value that the function returns.
